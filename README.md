### git

git配置以及命令行操作

1. 使用git diff > patch 生成补丁文件，patch可任意命名
2. 使用git apply --stat patch 查看补丁状态
3. 使用git apply --check patch 测试是否可以顺利接受补丁
4. 使用git apply --reject patch 将补丁先行打上，冲突的会生成.rej文件，需要手动处理
5. 使用git apply patch 接受补丁
